Kaleidoscope Project Applications
Downloadables

Both applications are distributed in the form of their installers assembled for Windows, and macOS computers. They can be downloaded from this page and placed on the hard disk following the standards of the named OSs. Installers having extension ".msi" are intended for installing and running applications on the Windows computers. Installers having extension ".pkg" are intended for installing and executing applications on the computers equipped with macOS.

All these installers, during the installation of their applications, ask:

1) to accept the End-User Licence Agreement; and

2) to either choose the Application Destination Folder, or install the app in the default location.

Each, the Windows OS and the MacOS, group of buttons located below suggests downloading the QDS MSE and SEM ABS App installers created for the corresponding OS.

Windows OS downloadables:
Windows Application: Kaleidoscope-QDS-MSE.msi, size: about 40 MB / App size on disk: about 90 MB
Windows Application: Kaleidoscope-SEM-ABS.msi, size: about 40 MB / App size on disk: about 99 MB
MacOS downloadables:
Mac OS Application: Kaleidoscope-QDS-MSE.pkg, size: about 40 MB / App size on disk: about 102 MB
Mac OS Application: Kaleidoscope-SEM-ABS.pkg, size: about 40 MB / App size on disk: about 100 MB

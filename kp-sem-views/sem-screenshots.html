The Space Exploration Mission // An Agent-Robot Behavior Simulator
(Kaleidoscope Project)

This page is a brief description of Space Exploration Mission - an acting Agent-Robot behavior simulator Java desktop application backed by some screenshots. The Application is imaginary Mission Control and Monitoring Center (MCMC) interacting with the remote Agent-Robot - a rover that may execute different perations. The task of the Agent is to explore the designated space, collect environmental parameters and transmit ita current coordinates packed with collected data to MCMC in accordance with scheduled exchange sessions. The Agent can be controlled manually - via manual controls, or automatically - by McLN-based Agent Behavior Controller. The task of the Agent Behavior Controller is to take input from Agent's sensors, recognize situations arising dynamically, make real-time control decisions, and activate relevant to situations actions by communicating with Agent's local controllers. Figure 1. Space Exploration Mission application screen snapshot.

Space Exploration Mission application screen is the imitation of the Mission Control and Monitoring Center dashboard. It consisting of following control and monitoring panels: (1) Manual Robot Control panel (left panel), (2) McLN-based Robot Behavior Controller (center panel), (3) Robot Observation Monitors - the cluster of monitors occupying right part of the screen, displaying a) the remote world and accomplishing its exploratory mission robot 3D view, b) robot 2D front and side views c) the robot's view - the world view as it is seeing from the cabin of the Rover, and (4) Running Text Panel - the panel located below the monitor panels presenting received from the Rover remote data.
